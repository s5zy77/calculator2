# Latestcalc2 (Sentinel Safety App)

A premium safety and calculator application frontend (`index.html`) working alongside a server (`server.js`) designed for real-time SOS features.

## Files
- `index.html`: The main UI of the app.
- `server.js`: The backend server for SMS gateway integration.
- `.env`: Environment variables (ignored in Git).
- `.gitignore`: Specifies intentionally untracked files like `.env` and `node_modules`.
